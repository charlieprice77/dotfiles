# dotfiles

Neovim + kitty config

## Installation
Clone this repo to your home directory

## DO NOT RUN SYNC

- ln -sf ~/dotfiles/nvim ~/.config/nvim
- ln -sf ~/dotfiles/kitty.conf ~/.config/kitty/kitty.conf
- Open nvim lazy will install plugins
- :MasonInstall clangd
- npm install -g tree-sitter-cli

## Done :)
